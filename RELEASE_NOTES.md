# Release Notes

## Version 1.0.0

Initial release of the Car Dealership Inventory System.

### Features

- User Registration
- User Login
- JWT Authentication
- Role-Based Authorization
- Vehicle CRUD
- Vehicle Search
- Purchase Vehicle
- Restock Vehicle
- MongoDB Integration
- React Frontend
- Bootstrap UI

### Improvements Planned

- Better Dashboard
- Responsive UI
- Pagination
- Vehicle Images
- Docker Deployment
- Automated Testing
